## 📋 Quick Start (Choose One)

### Option A: Edit CONFIG (Simplest)
```bash
# 1. Run setup
bash setup.sh

# 2. Edit CONFIG in script
nano tick_species_metagenome_identifier.py
# Change lines 30-50 with your files

# 3. Run
python tick_species_metagenome_identifier.py

# 4. Results
cat tick_analysis/species_identification_report.txt
```

### Option B: Use Command-Line (Best for Scripts)
```bash
python tick_species_metagenome_identifier.py \
  -r Tick_Specifier_Ticks.fasta \
  -i your_sample.fastq \
  -t 8
```

### Option C: CONFIG + Override (Testing)
```bash
# CONFIG has defaults
python tick_species_metagenome_identifier.py \
  --threads 16 \
  --min-identity 0.95
```
